üìä Menggunakan Dashboard E-Rapor
  • Pilih Mode Kerja
    Aplikasi memiliki 2 mode: Mode Rapor untuk mengelola data rapor, dan Mode Identitas Siswa untuk mengelola data identitas.
  • Atur Filter Data
    Pilih Tahun Pelajaran (2025/2026, 2026/2027, atau 2027/2028) dan Semester (Ganjil/Genap), lalu klik "Cek Status".
  • Periksa Status Ledger
    Sistem menampilkan kelengkapan rapor dan identitas siswa beserta persentasenya secara otomatis.
üìÇ Mengelola Data Siswa
  • Tambah Data Siswa
    Klik tombol "Tambah Siswa" di kanan atas tabel, isi formulir yang muncul dengan lengkap, lalu klik "Simpan".
  • Unggah File Ledger / Rapor
    Pada kolom Aksi di tabel siswa, klik "Upload", pilih file PDF yang sesuai, dan tunggu hingga status berubah menjadi "Tersedia".
